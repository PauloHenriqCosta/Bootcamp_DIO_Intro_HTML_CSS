Codar e ser Dev

Quando aprendi sobre a existência das tags comecei a entender a diferença entre codar e ser dev : numa primeira olhada está tudo ok mas ao analisar o código um site entrega maior valor enquanto outro "só funciona".

Essa conclusão serviu de alerta para eu tomar cuidado com o estilo de estudo. Caso eu me preocupasse somente com o funcionamento(sem entender a base), eu estaria treinando pra ser um eterno junior.

Considerando que a TI é feita de prática/teoria/prática e considerando ainda que quero o cargo de junior como algo temporário:

Criei esse site para praticar (e aprender enquanto pratico).
